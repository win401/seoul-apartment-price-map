# Portfolio Case Study: Seoul Apartment Price Map

## 한 줄 소개

서울 아파트 실거래 데이터를 분석해 3억 투자금을 가진 초보 부동산 투자자가 볼 수 있는 3년 후 가격 예측 지도 서비스를 만든 프로젝트입니다.

## Live

- Demo: https://nextjs-map-prototype.vercel.app
- GitHub: https://github.com/win401/seoul-apartment-price-map

## 문제 정의

부동산 데이터는 숫자와 표만으로는 초보자가 해석하기 어렵습니다.

이 프로젝트는 단순히 가격 예측 모델을 만드는 것에서 끝내지 않고, 사용자가 다음 질문에 답할 수 있는 화면을 만드는 것을 목표로 했습니다.

- 3억 이하 투자금으로 볼 수 있는 서울 아파트 후보는 어디인가?
- 2014년 기준 정보만으로 2017년 가격을 어느 정도 예측할 수 있는가?
- 예측 결과를 지도에 표시하면 후보 비교가 쉬워지는가?

## 페르소나

초보 부동산 투자자:

- 투자 가능 금액: 약 3억 원
- 관심 지역: 서울
- 원하는 정보: 현재 가격, 3년 후 예측 가격, 상승률, 검증 오차
- 어려움: 원본 데이터와 모델 지표만으로는 실제 후보를 고르기 어렵다

## 사용 데이터

- 수업 실습용 아파트 실거래 데이터
- 전체 학습 데이터: 1,216,553행
- 서울 학습 데이터: 742,285행
- 예측 대상: `transaction_real_price`
- 단위 해석: 만원 단위

## 분석 과정

1. 원본 데이터 구조 파악
2. 서울 데이터만 분리
3. 거래가 단위와 평당 단가 단위 해석
4. 평수, 평당 단가, 거래 연월, 건물 나이 파생 변수 생성
5. 거래가와 평당 단가 분포 시각화
6. 이상치 기준 정의
7. baseline 회귀 모델 학습
8. 3억 이하 투자자 페르소나 기준 후보 추출
9. 2014년 데이터로 2017년 가격을 예측하는 백테스트 설계
10. 후보 데이터를 지도 서비스용 JSON으로 변환
11. Kakao Local API로 주소 좌표 변환
12. Next.js + Leaflet 지도 화면 구현
13. GitHub 저장소 연결 및 Vercel 배포

## 모델 결과

Baseline 모델:

- 평균값 기준 모델 MAE: 약 2.2237억 원
- 릿지 회귀 모델 MAE: 약 1.6443억 원
- 수치형 부스팅 모델 MAE: 약 1.2186억 원

3년 후 예측 모델:

- 검증 방식: 2014년 데이터로 2017년 가격 예측
- 3억 이하 신뢰 후보: 1,292개
- 2014년 대비 2017년 상승률 중앙값: 약 26.35%
- MAE: 약 0.1854억 원
- RMSE: 약 0.2407억 원
- R2: 0.8718
- 상승/하락 방향 정확도: 98.76%

## 한계

이 프로젝트는 교육용/포트폴리오용 프로토타입입니다.

- 실제 투자 추천 서비스가 아닙니다.
- 2014년과 2017년 사이의 과거 데이터 검증 결과입니다.
- 금리, 정책, 학군, 교통, 공급량, 재건축 같은 외부 변수는 반영하지 않았습니다.
- 실시간 가격 예측 API는 아직 없습니다.
- 지도 화면 범위 기준 검색과 단지명 검색은 아직 구현하지 않았습니다.

## 다음 개선 방향

우선순위:

1. 지도 화면 범위 기준 후보 필터
2. 구/동/단지명 검색
3. 후보 즐겨찾기
4. 외부 변수 추가: 지하철, 공원, 학교, 보육시설
5. FastAPI 기반 실시간 예측 API
6. 포트폴리오 사이트에 프로젝트 카드 추가

## 이 프로젝트의 의미

이 프로젝트는 수업 실습 결과를 단순 노트북으로 끝내지 않고, 실제 사용자가 볼 수 있는 서비스 형태로 확장한 사례입니다.

취업 포트폴리오에서는 `데이터 분석 + 머신러닝 + 서비스 기획 + 프론트엔드 구현 + 배포`를 하나의 흐름으로 설명할 수 있습니다.

1인 기업 관점에서는 특정 도메인 데이터를 분석해 의사결정 도구로 만드는 실험 자산으로 활용할 수 있습니다.
