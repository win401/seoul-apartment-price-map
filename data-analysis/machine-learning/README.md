# Machine Learning

인공지능 사관학교 머신러닝 실습을 관리하는 폴더.

## Goal

수업 실습을 단순 저장하지 않고, 아래 두 방향으로 전환할 수 있게 정리한다.

- 취업 포트폴리오: 데이터 기반 문제 해결 사례
- 1인 기업 자산: SaaS, 자동화, 고객 분석, 추천/예측 기능의 기초

## Folder Rule

날짜별 실습은 아래 형식으로 만든다.

```text
YYYY-MM-DD-topic/
├── README.md
├── notebooks/
├── data/
│   ├── raw/
│   └── processed/
├── outputs/
└── reports/
```

## Practice Checklist

각 실습마다 최소한 아래 내용을 남긴다.

- 오늘 배운 개념
- 사용한 데이터
- 문제 정의
- 전처리 방식
- 사용한 모델
- 평가 지표
- 결과 해석
- 포트폴리오/사업화 가능성

## Portfolio Lens

좋은 실습은 나중에 아래 형식으로 정리한다.

- Problem: 어떤 문제를 풀었는가
- Data: 어떤 데이터를 사용했는가
- Method: 어떤 분석/모델링을 했는가
- Result: 어떤 결과가 나왔는가
- Insight: 비즈니스나 서비스에 어떤 의미가 있는가
- My Role: 내가 직접 한 일은 무엇인가
