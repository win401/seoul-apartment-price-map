# Class

광주 인공지능 사관학교 수업 자료와 과제를 관리하는 폴더.

## Purpose

이 폴더는 단순 수업 저장소가 아니다.

목표:

1. 인공지능 사관학교 수업을 체계적으로 관리한다.
2. 수업 과제 중 사업화 가능성이 있는 것을 `business-idea-lab`으로 연결한다.
3. 1인 기업 목표가 예상보다 늦어질 경우, 수업/개인 프로젝트를 취업 포트폴리오로 전환한다.

## Structure

- `data-analysis/`: 데이터 분석 수업, 실습, 리포트
  - `machine-learning/`: 머신러닝 실습, 노트북, 결과 정리
- `ai-service-planning/`: AI 서비스 기획 수업 자료
- `assignments/`: 과제 제출물
- `notes/`: 수업 노트
- `portfolio-candidates/`: 포트폴리오 후보

## Rule

수업 프로젝트는 처음에는 `class`로 관리한다.

쓸만해지면 아래 중 하나로 승격한다.

- `portfolio-candidates`: 취업 포트폴리오 후보
- `business-idea-lab/idea-backlog.md`: 사업 아이디어 후보
- 기존 프로젝트: `Kmong_project`, `roomy_project`, `job_simulator_pj` 등과 연결
